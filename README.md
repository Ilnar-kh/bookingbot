# Telegram Booking Bot — сервис онлайн-записи клиентов

Java, Spring Boot, PostgreSQL, Spring Data JPA, Hibernate, TelegramBots API, Lombok

## О проекте

**Telegram Booking Bot** — backend-приложение и Telegram-бот для автоматизации записи клиентов на услуги.  
Проект ориентирован на малый бизнес: салоны красоты, стоматологии, студии массажа и другие сервисные компании.

Бот позволяет клиентам самостоятельно выбрать дату и время записи, а администраторам — управлять временными слотами и просматривать записи.

## Основные возможности

- интерактивное бронирование с выбором даты через календарь;
- выбор времени из доступных слотов;
- пошаговая запись клиента с подтверждением;
- сбор имени и номера телефона;
- сохранение записей в базе данных;
- уведомления администратору о новых записях;
- админ-панель для управления слотами;
- изменение и удаление временных слотов;
- разграничение прав клиента и администратора.

## Архитектура

Проект реализован как монолитное backend-приложение с использованием Spring Boot и Telegram Bot API.

Основные компоненты:
- Bot — обработка команд и пользовательских состояний;
- Handlers — логика обработки шагов диалога;
- Services — бизнес-логика;
- Repositories — работа с базой данных;
- Session layer — хранение состояния пользователя в процессе записи.

## Административные функции

Администратор может:
- просматривать записи через календарь;
- видеть занятые и свободные слоты;
- редактировать и удалять временные слоты;
- просматривать данные клиента:
  - имя;
  - телефон;
  - услуга;
  - дата и время записи.

Доступ к административным функциям ограничен списком Telegram ID администраторов.

## Поддержка нескольких услуг

Проект легко расширяется под разные бизнес-сценарии:
- добавление нескольких услуг;
- разделение временных слотов по специалистам;
- разные сценарии записи в зависимости от выбранной услуги.
